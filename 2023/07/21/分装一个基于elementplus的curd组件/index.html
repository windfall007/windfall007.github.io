分装一个基于element-plus的curd组件

分类： 前端
标签： vue

日期：2023-07-21 15:20:54

公司写业务的时候发现AVUE这个框架很好用，但项目已经写了一部分了，于是参照着AVUE实现了一个基础的curd组件。使用json配置的方式，是一个很好的编程思维，写一个说明文档，方便以后查阅和修改组件。

使用说明

使用组件需要项目安装element-plus ui 库^2.3.8以上版本，vue ^3.2.36 以上版本。使用一下版本可能会报版本不兼容的错误。

curd组件示例

HTML部分

<curd
   ref="curdModel"
       v-model:page="page"
       :data="tableData"
       :options="options"
       @row-save="rowSave"
       @row-update="rowUpdate"
       @row-del="rowDel"
       @refresh-change="refreshChange"
       @search-change="searchChange"
   >
       <template #status="scope">
           <el-tag v-if="scope.row.status ==2" type="success">在线</el-tag>
           <el-tag v-else-if="scope.row.status ==1" type="danger">离线</el-tag>
       </template>
       <template #actions="scope">
       <!--自定义操作-->
       <el-button type="primary" size="mini" :icon="Tickets">详情</el-button>
       </template>
   </curd>

Attributes

属性名 说明 类型 可选值 默认值
options 表单配置项参考Option配置 object - -
data 显示的数据 array - -
v-model:page 分页变量参考Page参数 object - -
disableHeight table自适应高度 Boolean true，false false

Events

事件名 说明 回调参数
row-save 新增数据后点击确定触发该事件 res,done
row-update 数据更新触发事件 res,done
row-del 数据删除触发事件 res,done
refresh-change 数据刷新触发事件 res,done
search-change 搜索按钮触发 res,done

res 返回当前数据，done完成触发

options

参数 说明 类型 可选值 默认值
actions curd组件增删改查操作 Array edit/del/add/refresh/filter/layout/fullScreen/pagination -
column table column字段 Array - -

options-actions

参数 说明 类型 可选值 默认值
del 数据删除 String/Object - del
add 数据增加按钮 String/Object - add
edit 数据编辑按钮 String/Object - add
refresh 刷新按钮 String - refresh
filter 过滤按钮 String - filter
layout 布局按钮 String - layout
fullScreen 全屏按钮 String - fullScreen
pagination 分页组件 String - pagination

del 参数的自定义模式

{ 
	name: 'del',
	text: '解绑', //定义按钮名称
    icon:''//按钮图标
    size:'large/small/default'//按钮大小
    plain:
   confirm: {
   	  //参数提示
     message: '是否要解绑当前网关？，解绑后和该网关绑定的设备将移除',
     //删除按钮文字
     confirmButtonText: ''
   }
}

options-column

参数 说明 类型 可选值 默认值
prop 列标题的内容属性名称 string - -
minWidth 对应列的最小宽度 Number - -
label 列标题 string - -
showOverflowTooltip 溢出宽度文字隐藏 true false/true false
align 文字对齐 String left/right left
